# SkillCraft Task 01 - Interactive Navigation Menu 🚀

## 📌 Project Overview

This project is developed as part of the **SkillCraft Web Development Internship - Task 01**.

The objective of this task is to create an **interactive navigation menu** that remains fixed on the webpage and changes its appearance based on user interaction such as scrolling and hovering.

## ✨ Features

* ✅ Fixed navigation bar visible while scrolling
* ✅ Navbar style changes when scrolling
* ✅ Smooth hover effects on menu items
* ✅ Responsive design for mobile devices
* ✅ Hamburger menu for smaller screens
* ✅ Smooth scrolling navigation
* ✅ Clean and modern UI design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Font Awesome Icons

## ⚙️ How It Works

* The navigation bar is positioned using CSS `position: fixed`.
* JavaScript detects scrolling events and changes the navbar style.
* CSS hover effects provide interactive feedback when users move over menu items.
* A responsive hamburger menu improves usability on mobile screens.

## 🎯 Learning Outcomes

Through this task, I practiced:

* Creating responsive navigation layouts
* Working with CSS transitions and animations
* Handling DOM events using JavaScript
* Improving user experience through interactive UI elements
