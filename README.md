# SkillCraft Task 01 - Interactive Navigation Menu

## 📌 Project Overview

This project is developed as part of the **SkillCraft Web Development Internship - Task 01**.

The objective of this task is to create an interactive navigation menu that remains fixed on the webpage and changes its appearance based on user interactions such as scrolling and hovering.

## ✨ Features

- Fixed navigation bar visible while scrolling
- Navbar style changes on scroll
- Smooth hover effects on menu items
- Responsive design for mobile devices
- Mobile hamburger menu
- Smooth scrolling navigation
- Clean and modern UI design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons

## ⚙️ How It Works

- The navigation bar is fixed using CSS positioning.
- JavaScript detects scrolling events and updates the navbar style.
- CSS hover effects provide interactive feedback for menu items.
- Responsive design ensures usability across different devices.

## 🎯 Learning Outcomes

Through this task, I practiced:

- Creating responsive navigation layouts
- Implementing CSS transitions and hover effects
- Handling DOM events using JavaScript
- Building interactive user interfaces
